## How this documentation stays current

This bundle is generated by `/document`. The knowledge-graph mechanics run through
`docs/ai-forward-pack/scripts/docs-graph.py` (V18): `derive` regenerates `docs-index.js` from
frontmatter, `freshness --gate` is the CI-able staleness check, and `snapshot` records a
graph-health trend. The freshness contract is documented in the architecture overview's
*Status & next action*.
